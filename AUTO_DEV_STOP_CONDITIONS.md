# 黔行智导 — 全项目自动开发总控协议 · 停止条件

> 本文件定义自动开发必须立即停止并等待用户介入的所有场景。
> 遇到任何停止条件时，自动开发不得自行决策绕过。

---

## 一、立即停止条件（CRITICAL）

以下情况一旦发生，自动开发**立即停止**，不允许任何形式的自动绕过：

### 安全相关

| # | 停止条件 | 说明 |
|---|---------|------|
| S1 | 需要真实 API Key / Token / 密钥 | 包括 AI API、天气 API、地图 API、支付密钥等 |
| S2 | 需要真实数据库账号密码 | 无法使用本地开发数据库时 |
| S3 | 需要服务器 SSH 账号或部署权限 | 部署阶段 |
| S4 | 检测到敏感信息泄露 | 密钥、Token、密码出现在代码中 |
| S5 | 需要接入第三方支付/短信/认证服务 | 需要商户号、短信模板、OAuth 配置等 |

### 数据相关

| # | 停止条件 | 说明 |
|---|---------|------|
| D1 | 需要删除数据库/表/数据 | 包括 DROP DATABASE、TRUNCATE、DELETE 全表 |
| D2 | 需要修改已有 storage key 名称 | 会影响已有用户数据 |
| D3 | 需要修改数据结构且影响旧数据兼容 | 包括字段重命名、类型变更、删除必填字段 |
| D4 | 需要迁移大量已有数据 | 超过 10 条记录的批量修改 |

### 产品方向

| # | 停止条件 | 说明 |
|---|---------|------|
| P1 | 需要大幅改变主流程 | 影响 14 步演示流程中 3 步以上 |
| P2 | 需要修改 tabBar 结构 | 增减 tab、修改页面路径 |
| P3 | 需要新增/删除页面 | 改变页面数量或路径 |
| P4 | 需要新增付费/登录/定位/地图能力 | 超出当前产品范围 |
| P5 | 需要用户确认 UI 大方向 | 整体风格变更、页面重设计 |
| P6 | 需要用户确认图片/素材选择 | 新景点图、品牌素材、空态插画 |

### 技术阻塞

| # | 停止条件 | 说明 |
|---|---------|------|
| T1 | P0 问题经 2 轮修复仍无法解决 | 修复循环上限 |
| T2 | P1 问题经 2 轮修复仍存在 | 修复循环上限 |
| T3 | 编译错误超过 30 分钟无法定位 | 根因不明的编译错误 |
| T4 | git 冲突（merge/rebase conflict） | 需要人工解决 |
| T5 | 需要修改项目技术栈 | 框架升级、新语言、新构建工具 |
| T6 | 需要安装新的大型依赖 | 如 ffmpeg、数据库服务、Redis |
| T7 | 微信开发者工具不可用 | 无法验证小程序编译 |
| T8 | npm/node 环境异常 | 无法运行构建 |

### 文件操作

| # | 停止条件 | 说明 |
|---|---------|------|
| F1 | 需要删除 5 个以上文件 | 批量删除 |
| F2 | 需要 force push | 修改 git 历史 |
| F3 | 需要修改 CLAUDE.md 中的核心规则 | 产品外显原则、赛题对齐、禁止事项 |
| F4 | 需要修改 FINAL_DELIVERY_CHECKLIST.md | 交付清单由人工审核 |

## 二、暂停条件（可自动恢复）

以下情况暂停自动开发，等待条件满足后自动恢复：

| # | 暂停条件 | 自动恢复条件 |
|---|---------|------------|
| W1 | 文件被外部进程锁定 | 锁定解除 |
| W2 | 网络临时不可用（获取资源时） | 网络恢复 |
| W3 | 微信开发者工具正在编译 | 编译完成 |

暂停等待最长 5 分钟，超时则升级为停止条件。

---

## 三、停止后的操作流程

当自动开发因停止条件中断时：

1. 在 AUTO_DEV_STATE.md 设置 `stop_required: true` + `stop_reason: <具体原因>`
2. 在 AUTO_DEV_REPORT.md 记录停止信息
3. 通过 PushNotification 通知用户（如可用）
4. 输出清晰的停止报告，包含：
   - 停止原因
   - 当前完成进度
   - 需要用户做什么
   - 建议的下一步

---

## 四、用户恢复指令

用户可通过以下方式恢复自动开发：

- "继续自动开发" → 从当前停止点继续
- "跳过当前阶段，进入下一阶段" → 标记当前阶段为 skipped，进入下一阶段
- "停止自动开发，切换为手动模式" → 退出自动开发模式
