# 黔行智导 · 小程序核心流程联调清单

## 前置条件
- 后端已启动 (localhost:8080)
- 管理员账号已初始化
- DevTools 已导入根目录
- 已勾选"不校验合法域名"

## 核心流程
- [ ] 首页 → 打开正常，景区列表加载
- [ ] 首页 → 景区选择卡片可点击
- [ ] 景区详情 → 景点信息/路线/天气参考展示
- [ ] AI助手 → 进入正常，快捷问题列表
- [ ] AI助手 → 点击"贵州必去景点"有回答
- [ ] AI助手 → 点击"今天怎么玩"弹出位置弹窗
- [ ] 位置弹窗 → "暂不使用"仍返回建议
- [ ] 位置弹窗 → "使用当前位置"正常
- [ ] AI草稿 → 点击"生成行程攻略"生成草稿
- [ ] AI草稿预览 → 打开正常，展示日程
- [ ] AI草稿 → "保存到我的行程"成功
- [ ] 我的行程 → 列表显示已保存行程
- [ ] 我的行程 → 点击进入行程详情
- [ ] 路线详情 → 天气参考卡片
- [ ] tabBar → 首页/AI助手/我的行程切换正常
- [ ] 页面间跳转 → 不白屏，数据正确

## 异常流程
- [ ] 后端未启动 → 网络错误友好提示
- [ ] 登出 → 重新登录流程
- [ ] 草稿失效 → "草稿已失效请重新生成"
- [ ] 无行程 → "今天怎么玩"给出通用建议
